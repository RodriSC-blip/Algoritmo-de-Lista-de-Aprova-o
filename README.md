O código em questão serve para armazenamento rápido de dados de alunos, sendo ultil à area, para adicionar a um sistema de médias.
No momento o programa aceita: Matricula, Nome de Alunos, 2 notas de 1 a 10 que serão usadas para calculo de média e  visualização da situação do aluno,
aprovado ou reprovado, tendo um valor minimo de sete (7) para considerar aprovado.
